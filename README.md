# P1: Data Detective

## Summary
This project analyzes a French literary text file, counts word frequencies, displays the top N most common words, and provides an additional insight into the dataset.

## Dataset
- File: `data/boylesve_reflexions.txt`
- Why I chose it:  
  I chose *Réflexions sur Stendhal* by René Boylesve because it is a rich literary text with meaningful vocabulary and stylistic depth. It allows for interesting analysis of word frequency and language patterns. Additionally, working with a French text introduces accents and unique structures, making the processing more challenging and realistic.
